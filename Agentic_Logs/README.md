### Uso de este fichero

Este fichero será usado por cada miembro del equipo para almacenar de forma local todas las conversaciones con LLMs para posteriormente documentarlas como se indicó al inicio de la asignatura.

### ¿Por que está en el gitignore?

Por que las conversaciones pueden ser muy *variadas* y tomar tonos extraños que por privacidad preferimos que no sean públicas para todo el mundo en GitHub, todas las conversaciones serán debidamente documentadas para la entrega a los profesores y únicamente a ellos.

### Formato

Las conversaciones se almacenarán con el siguiente formato DD-MM-YYYY-XXX.md, siendo XXX el número de la conversación en el día, de esta manera podemos saber cuantos cambios de contexto ha habido. Los archivos tendrán la conversación completa con todo el contexto de inicio a fin (en la medida de lo razonablemente posible, es plausible que el código leido por el agente en esa iteración no se incluya debido a la dificultad de saber que está leyendo). Además las conversaciones se presentarán en un formato estandarizado entre todos los miembros.

El formato estandarizado dependerá de la aplicación / UI / Wrapper usado a la hora de hacer las peticiones debido a diferencias en el manejo del estado.
